# LearnMeNow – Backend API

LearnMeNow is an educational platform designed to support students preparing for various competitive exams. This backend service powers the core functionality, including user management, authentication, course access, progress tracking, and more.

## 🛠 Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB + Mongoose**
- **JWT Authentication**
- **Joi Validation**
- **Multer** (if file uploads are supported)
- **Socket.IO** (if using real-time chat)
- **Stripe** (if using subscriptions/payments)
